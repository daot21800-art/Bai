Trang 1/11

1. Khi nào nên dùng hàm?

Tái sử dụng

Tránh viết lại cùng một đoạn mã ở nhiều nơi trong chương trình.

Chia nhỏ bài toán

Giúp chương trình dễ quản lý, bảo trì và kiểm soát lỗi hiệu quả hơn.

Tính đóng gói

Giấu đi sự phức tạp bên trong, chỉ quan tâm đến đầu vào và kết quả.

Trang 2/11

2. Cú pháp tạo hàm

CẤU TRÚC CHUẨN 
<Kiểu_trả_về> TenHam(<Tham_số>) {
    // Các câu lệnh xử lý
    return <Giá_trị>; 
}

Nếu kiểu trả về là void, hàm không cần lệnh return giá trị.

  • Kiểu_trả_về: int, float, char, void...
  • Tên_hàm: Nên đặt theo động từ (camelCase).
  • Tham_số: Các biến nhận dữ liệu đầu vào.
Trang 3/11

3. Ví dụ: Hàm có kiểu trả về

Hàm trả về kết quả để sử dụng cho các tính toán khác.

int tinhDienTich(int dai, int rong) {
    int s = dai * rong;
    return s; // Trả về diện tích
}

→ Kết quả này có thể gán vào biến hoặc in ra màn hình.

Trang 4/11

4. Ví dụ: Hàm không trả về (void)

Hàm thực hiện một tác vụ cụ thể và kết thúc ngay sau đó.

void thongBaoLoi() {
    printf("Lỗi: Giá trị nhập vào phải > 0!\n");
}

→ Thường dùng để in thông báo, vẽ giao diện, hoặc cập nhật trạng thái.

Trang 5/11

5. Cách gọi hàm void

Gọi trực tiếp như một câu lệnh độc lập:

TenHam(doi_so);

Ví dụ thực tế:

thongBaoLoi();
Trang 6/11

6. Cách gọi hàm có giá trị trả về

  • Gán vào biến: int dt = tinhDienTich(5, 4);
  • Dùng trong biểu thức: int total = tinhDienTich(5, 4) + 10;
  • In trực tiếp: printf("S = %d", tinhDienTich(5, 4));
Trang 7/11

7. Biến và Tầm vức (Scope)

Biến toàn cục

Khai báo ngoài mọi hàm. Có thể truy cập từ bất cứ đâu trong chương trình.

Biến cục bộ

Khai báo trong { }. Chỉ hàm đó mới thấy và sử dụng được biến này.

Trang 8/11

8. Tham số vs Đối số

Tham số (Parameter)

Biến khai báo ở tiêu đề hàm.

void func(int n); // 'n' là tham số

Đối số (Argument)

Giá trị thực truyền vào khi gọi.

func(10); // '10' là đối số
Trang 9/11

9. Các cách truyền tham số

Cách truyền Cú pháp Biến gốc sau hàm Bản chất
Tham trị void tang(int x) Không đổi Hàm dùng bản sao giá trị.
Địa chỉ void tang(int *x) Bị thay đổi Hàm tác động trực tiếp vào ô nhớ.
